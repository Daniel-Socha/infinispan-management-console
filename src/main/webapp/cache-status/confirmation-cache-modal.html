Confirmation

Enabling cache '{{cacheName}}' will attach it to all active remote endpoints.

Disabling cache '{{cacheName}}' will detach it from all active remote endpoints.

Clearing cache '{{cacheName}}' will remove all of its cache entries both from memory and from any associated persistent stores.

This operation is destructive and cannot be undone !

Flushing cache '{{cacheName}}' will passivate all its cache entries.

Re-indexing cache '{{cacheName}}' will remove its current index and recreate it according to the cache contents' schema.

Enabling rebalancing of '{{cacheName}}' will automatically redistribute data across the cluster when nodes are added/removed.

Disabling rebalancing of '{{cacheName}}' will NOT automatically redistribute data across the cluster when nodes are added/removed.
